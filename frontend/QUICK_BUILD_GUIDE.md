# 📦 Быстрая сборка APK для Google Play Market

## 🚀 Самый простой способ (EAS Build - облачная сборка)

### 1. Установите EAS CLI
```bash
npm install -g eas-cli
```

### 2. Авторизуйтесь в Expo
```bash
eas login
```
Зарегистрируйтесь на https://expo.dev если у вас нет аккаунта.

### 3. Запустите сборку Production APK
```bash
cd /app/frontend
eas build --platform android --profile production
```

**Готово!** Сборка займёт 10-20 минут. После завершения вы получите ссылку на скачивание APK.

---

## 🎬 Или используйте скрипт для автоматизации

```bash
cd /app/frontend
./build.sh
```

Выберите:
1. Preview APK - для тестирования
2. Production APK - готовый APK для Play Market
3. Production AAB - оптимизированный формат для Play Market (рекомендуется)

---

## 📱 Что дальше?

После получения APK/AAB:

1. **Протестируйте APK** на физическом устройстве
2. **Зарегистрируйтесь** в Google Play Console (https://play.google.com/console)
3. **Создайте новое приложение**
4. **Загрузите APK/AAB**
5. **Заполните описание и скриншоты**
6. **Отправьте на модерацию**

Модерация обычно занимает 1-7 дней.

## 🆘 Нужна помощь?

**Распространённые вопросы:**

**Q: EAS Build требует платную подписку?**
A: Нет, EAS Build предоставляет бесплатные сборки (с ограничением по времени ожидания в очереди).

**Q: Можно ли собрать без облака?**
A: Да, смотрите "Вариант 2: Локальная сборка" в BUILD_INSTRUCTIONS.md

**Q: Сколько стоит публикация в Play Store?**
A: Google Play требует единоразовый взнос $25 для создания аккаунта разработчика.

**Q: Нужен ли keystore?**
A: EAS Build автоматически создаст и сохранит keystore. При локальной сборке создайте самостоятельно.
