# Kraken Messenger

## Project Overview
Kraken Messenger is a robust messaging application designed for secure and efficient communication. With a focus on user privacy and data security, Kraken Messenger offers a variety of features to enhance user interaction and ensure reliable messaging.

## Installation Instructions  
To install Kraken Messenger, follow these steps:  
1. Clone the repository:  
   ```bash
   git clone https://github.com/kotogum-kg/Kraken-Messenger.git
   ```  
2. Change directory to the project folder:  
   ```bash
   cd Kraken-Messenger
   ```  
3. Install the required dependencies:  
   ```bash
   npm install
   ```  
4. Start the application:  
   ```bash
   npm start
   ```

## Features
- Secure communication with end-to-end encryption  
- User-friendly interface  
- Group chat functionality  
- File sharing capabilities  
- Real-time messaging  
- Push notifications
